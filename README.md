# MyChunav

A community platform to create, discover, vote on, and rank anything — food, movies, sports, brands, cities, companies, and public issues. India's collective opinion, in one place.

## Tech Stack
- **Next.js** (React framework, frontend + backend in one project)
- **TypeScript** (safer JavaScript with type-checking)
- **Tailwind CSS** (utility-first styling)
- **PostgreSQL** via Prisma (coming in Phase 2)
- **NextAuth.js** (coming in Phase 3)
- **Vercel** (deployment, Phase 7)

## Getting Started Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
